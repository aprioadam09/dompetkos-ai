# DompetKos AI

DompetKos AI is an Indonesian-language LLM-based personal finance assistant for students and boarding house residents.

The project focuses on extracting structured financial transactions from natural language input, validating the extracted data, storing it in a local database, and providing simple spending insights through a dashboard.

## MVP Scope

- Natural language transaction input in Bahasa Indonesia
- LLM-based structured JSON extraction
- Validation layer for amount, date, category, and transaction type
- SQLite database for transaction storage
- Monthly budget tracking
- Basic spending analytics
- Streamlit dashboard

## Tech Stack

- Python
- Streamlit
- SQLite
- Gemini API
- Pydantic
- Pandas
- Plotly
